Chapter 4 Terminology
Term Definition
Network Classifications:
  • LAN - Interconnects computers within a limited area.
  • MAN - Interconnects users in a region the size of a metropolitan area.
  • WAN - Type of telecommunications network that spans a large geographical region.
Network Models:
  • Bus network - Is a network topology in which nodes are directly connected to a common half-duplex link.
  • Star network - Is a LAN in which all nodes are directly connected to a common central computer.
Protocols:
  • CSMA/CD (Ethernet) - Carrier-sense multiple access with collision detection uses carrier-sensing to avoid collisions in Ethernet networks.
  • CSMA/CA (Wifi) - Carrier-sense multiple access with collision avoidance senses the channel's availability/avoids collisions by transmitting when it's idle.
Combining Networks:
  • Repeater - Regenerates and retransmits incoming signals in a network.
  • Bridge - Is designed to connect networks.
  • Switch - Channels incoming data from multiple input ports to an output port.
  • Router - Is a device that connects two or more packet-switched networks or subnetworks.
  • Gateway - Converts information, data or other communications from one protocol or format to another.
Client/Server Model:
  • P2P (Peer-to-Peer) - A distributed application architecture that partitions tasks or workloads between peers.
  • Distributed System - A collection of independent components located on different machines.
  • Cloud Computing - On-demand availability of computer system resources.
Internet Addressing:
  • Tier 1 - Internet Protocol (IP) network that can reach every other network on the Internet.
  • Tier 2 - An Internet service provider (ISP) which engages in the practice of peering with other networks.
  • Tier 3 - Primarily engaged in delivering Internet access to end customers.
IP Address:
  • ICANN - Internet Corporation for Assigned Names and Numbers.
  • DNS - Domain Name System is a hierarchical and distributed naming system.
  • TLD (Top Level Domain) - One of the domains at the highest level in the hierarchical Domain Name System.
  • Country-Code TLDs - An Internet top-level domain generally used or reserved for a country.
Internet Protocols:
  • FTP (File Transfer Protocol) - Is a standard network protocol used for the transfer of files.
  • HTTP (Hypertext Transfer Protocol) - Is an application layer protocol in the Internet protocol suite model.
  • SMTP (Simple Mail Transfer Protocol) - Is an Internet standard communication protocol for electronic mail transmission.
World Wide Web:
  • URL (Uniform Resource Locator) - A web address that specifies the location of a resource on the internet, typically a webpage.
  • Client-Side Activities - Refers to activities or processes that take place on the user's device or web browser, such as running scripts or executing code.
  • Server-Side Activities - Any activities or processes that take place on the server, such as processing and storing data, handling requests, and generating responses.
Internet Four-Layer Software:
  • Application Layer - Responsible for providing services to user applications and enabling them to communicate over a network.
  • Transport Layer - Responsible for ensuring reliable transmission of data between end systems.
  • Network Layer - Responsible for addressing, routing, and forwarding data packets between different networks.
  • Link Layer - Responsible for providing a reliable, error-free communication channel between two adjacent nodes on a network.
Internet Is:
  • Packet-Switching Network - A type of network in which data is transmitted in packets from one node to another, with each packet being independently routed based on its destination address.
  • Fault Tolerant - Refers to a system or network that can continue to function even if one or more components fail.
  • Redundant - Refers to the use of extra or duplicate components in a system to ensure reliability and fault tolerance.
Cybersecurity:
  • Malware - Short for malicious software, refers to any type of software that is designed to harm or exploit computer systems, networks, or devices.
  • Virus - A type of malware that is capable of self-replicating and infecting other files or systems. Viruses typically require human action to spread, such as opening an infected file or clicking on a malicious link.
  • Worm - A self-replicating type of malware that spreads through networks and can cause significant damage by consuming network bandwidth or compromising system resources.
  • Trojan Horse - A type of malware that is disguised as legitimate software, but contains malicious code that can harm or compromise a system. Trojan horses often require user action, such as downloading or installing a fake software update.
  • Spyware - A type of malware that is designed to secretly monitor and collect information from a system or device. Spyware can capture sensitive information such as passwords, keystrokes, and browsing history.
  • Phishing - A type of social engineering attack that involves the use of fraudulent emails, websites, or messages to trick users into divulging sensitive information, such as usernames and passwords.
  • DDoS Attack - Short for Distributed Denial of Service attack, is a type of attack that floods a system or network with traffic, rendering it unavailable to users.
Protection:
  • Firewall - A type of security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
  • Spam Filter - A software program or service that is designed to identify and filter out unwanted or unsolicited email messages, also known as spam.
  • Proxy Server - A type of server that acts as an intermediary between client devices and other servers or resources on a network.
  • Antivirus Software - A type of software designed to detect, prevent, and remove malware from computer systems and networks.
  • Cryptography - The practice of securing information by encoding it in a way that makes it unreadable to unauthorized users.
  • Public Key Encryption - A type of encryption that uses two keys, a public key and a private key, to encrypt and decrypt data.
