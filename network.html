Chapter 4 Terminology
Term Definition
Network Classifications:
  • LAN - Interconnects computers within a limited area.
  • MAN - Interconnects users in a region the size of a metropolitan area.
  • WAN - Type of telecommunications network that spans a large geographical region.
Network Models:
  • Bus network - Is a network topology in which nodes are directly connected to a common half-duplex link.
  • Star network - Is a LAN in which all nodes are directly connected to a common central computer.
Protocols:
  • CSMA/CD (Ethernet) - Carrier-sense multiple access with collision detection uses carrier-sensing to avoid collisions in Ethernet networks.
  • CSMA/CA (Wifi) - Carrier-sense multiple access with collision avoidance senses the channel's availability/avoids collisions by transmitting when it's idle.
Combining Networks:
  • Repeater - Regenerates and retransmits incoming signals in a network.
  • Bridge - Is designed to connect networks.
  • Switch - Channels incoming data from multiple input ports to an output port.
  • Router - Is a device that connects two or more packet-switched networks or subnetworks.
  • Gateway - Converts information, data or other communications from one protocol or format to another.
Client/Server Model:
  • P2P (Peer-to-Peer) - A distributed application architecture that partitions tasks or workloads between peers.
  • Distributed System - A collection of independent components located on different machines.
  • Cloud Computing - On-demand availability of computer system resources.
Internet Addressing:
  • Tier 1 - Internet Protocol (IP) network that can reach every other network on the Internet.
  • Tier 2 - An Internet service provider (ISP) which engages in the practice of peering with other networks.
  • Tier 3 - Primarily engaged in delivering Internet access to end customers.
IP Address:
  • ICANN - Internet Corporation for Assigned Names and Numbers.
  • DNS - Domain Name System is a hierarchical and distributed naming system.
  • TLD (Top Level Domain) - One of the domains at the highest level in the hierarchical Domain Name System.
  • Country-Code TLDs - An Internet top-level domain generally used or reserved for a country.
Internet Protocols:
  • FTP (File Transfer Protocol) - Is a standard network protocol used for the transfer of files.
  • HTTP (Hypertext Transfer Protocol) - Is an application layer protocol in the Internet protocol suite model.
  • SMTP (Simple Mail Transfer Protocol) - Is an Internet standard communication protocol for electronic mail transmission.
