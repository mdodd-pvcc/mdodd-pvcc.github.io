Operating Systems Terminology
Term Definition
Operating System Software that supports a computer's functions, such as scheduling tasks, executing applications, and controlling peripherals.
Job Queue Contains an ordered list of jobs waiting to be processed by a subsystem.
FIFO "First In, First Out" a method of organizing, handling, and manipulating the data structure of elements in a computing system.
Interactive Processing Allows the user to interact directly with the computer.
Time-Sharing CPU scheduling and multi-programming to provide each user with a small portion of a shared computer at once.
Multiprogramming The capablity of running multiple programs on a single processor.
Application Software Performs a specific function directly for an end user or, in some cases, for another application.
System Software Designed to provide a platform for other software.
Utility Software Systems and programs on a computer system that maintain its functionality.
Graphical User Interface (GUI) A form of user interface that allows users to interact with electronic devices through graphical icons and audio indicators.
Shell A computer program that exposes an operating system's services to a human user or other programs.
Kernel A computer program at the core of a computer's operating system and generally has complete control over everything in the system.
Device Driver A special kind of software program that controls a specific hardware device attached to a computer.
File Manager A computer program that provides a user interface to manage files and folders.
Memory Manager A software utility that operates in conjunction with the operating system.
Virtual Memory A section of a hard disk that's set up to emulate the computer's RAM.
Boot Strapping The program that initializes the operating system (OS) during startup.
ROM "Read-only memory", a type of computer storage containing non-volatile, permanent data that can only be read.
Deadlock A situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resource, resulting in both programs ceasing to function.
Spooling A specialized form of multi-programming for the purpose of copying data between different devices.
Super User (Or Administrator) A special user account used for system administration.
Multi-Factor Authentication An authentication method that requires the user to provide two or more verification factors to gain access to a resource.
