All About GitHub

  • Q What is a version control system?

  • A. It is a class of systems that manage changes to programs.

  • Q What is Git?

  • A. Git is a version control system.

  • Q What is Github?

  • A. Github is a host service for software development using Git.

  • Q What is the difference betwen Git and Github?

  • A. Git is a tool and Github is a repository.

  • Q Who started Github and how was it started?

  • A. Chris Wanstrath and PJ Hyett started Github, it started after they needed the ability to collaborate on projects.

  • Q What company owns it now?

  • A. Microsoft Corporation.

  • Q How much does a Github account cost?

  • A. Opening an account for individual usage is free, however you can purchase various add-ons.

  • Q What is the octocat?

  • A. Octocat is the mascot of Github made by Cameron McEfee.
